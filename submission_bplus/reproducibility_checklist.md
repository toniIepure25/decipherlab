# Reproducibility Checklist

## Main Commands

Run from the repository root.

1. Rebuild the real grouped replication pack:

```bash
python3 scripts/run_real_grouped_replication_pack.py
```

2. Rebuild the redesigned real downstream pack:

```bash
python3 scripts/run_real_downstream_redesigned_pack.py
```

3. Rebuild the bridge summary:

```bash
python3 scripts/build_real_vs_synthetic_bridge.py
```

4. Rebuild the propagation analysis:

```bash
python3 scripts/build_propagation_analysis.py
```

5. Rebuild the B-level hardening pack:

```bash
python3 scripts/build_blevel_hardening_pack.py
```

## Expected Main Outputs

- `outputs/real_grouped_replication_summary.csv`
- `outputs/real_downstream_redesigned_summary.csv`
- `outputs/real_vs_synthetic_bridge_summary.csv`
- `outputs/propagation_model_summary.csv`
- `outputs/propagation_thresholds.csv`
- `outputs/statistical_robustness_summary.csv`
- `outputs/real_data_trust_summary.csv`

## Seeds And Determinism

- The branch relies on deterministic manifests and fixed experiment configs already checked into the repository.
- Bootstrap summaries in the hardening scripts use fixed internal seeds.

## Data Dependencies

- Historical Newspapers processed grouped data under `data/processed/historical_newspapers_grouped_words/`
- ScaDS.AI processed grouped data under `data/processed/scadsai_grouped_words/`
- Historical gold-style annotations in `data/processed/historical_newspapers_grouped_words/gold_annotations.csv`

## Manual Steps

- No manual step is required to rebuild the final hardening summaries if the processed manifests and gold-style annotations already exist.
- Repeating the visual audit itself is manual and is not regenerated by script.

## Validation

```bash
python3 -m py_compile $(find src tests scripts -name '*.py')
python3 -m pytest --maxfail=1
```
