# New Claims Audit

## Target Claim If The New Branch Succeeds

The strongest credible claim for the next paper would be:

> Structured transcription uncertainty, when decoded with explicit sequence constraints, improves sequence-level recovery under ambiguity on synthetic-from-real glyph benchmarks.

## Claims That Would Be Supported Only If New Experiments Succeed

- sequence-level decoding converts some symbol-level rescue into measurable sequence gains
- conformal prediction sets offer safer or more stable downstream behavior than raw truncation alone
- historically grounded glyph corpora benefit differently from modern handwritten-symbol corpora under structural decoding

## Claims Explicitly Out Of Scope

- semantic decipherment
- plaintext recovery
- broad historical generalization
- “solving” undeciphered manuscripts
- universal calibration or conformal superiority on every dataset

## Evidence Required Before Making The New Claim

- consistent sequence-level improvements over greedy collapse on at least one benchmark family
- ambiguity-sweep evidence showing when the gains appear
- failure analysis showing where structural decoding does not help
- reliability reporting for coverage and set size, not only accuracy
