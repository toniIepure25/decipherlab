# Code of Conduct

DecipherLab is intended to be a respectful, professional, and inclusive research project.

## Expected Behavior

- Be constructive and courteous.
- Give feedback on ideas, not people.
- Keep communication relevant and specific.
- Respect differing viewpoints and levels of experience.

## Unacceptable Behavior

- Harassment, intimidation, or discriminatory language.
- Personal attacks, deliberate misrepresentation, or trolling.
- Publishing private information without consent.
- Spam or repeated disruptive behavior.

## Enforcement

Project maintainers may remove, edit, or close content that violates this code of conduct and may restrict participation when needed to protect the community.

## Reporting

If you need to report a concern, open a private security advisory or contact the maintainers through the repository's private GitHub channels.
